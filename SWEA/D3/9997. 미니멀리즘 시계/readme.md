0부터 360까지 수가 들어온다. 그러면 이는 0시부터 12시까지의 각도이다.
각 시는 30도를 가진다. 그리고 그 30도 안에서 60분이 할당된다.
따라서 수가 주어지면, 30도 단위로 시를 구하고, 나머지를 활용해서 분을 구한다.
예를 들어 111이라면, 30으로 나누면 몫이 3이다. 따라서 3시가 된다. 나머지 값은 21이다. 2를 곱해주어 42분의 값을 갖는다.
